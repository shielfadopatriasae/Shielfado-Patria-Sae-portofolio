# Portfolio Framework – IT Support (Light Tech UI + RTCCO)

## 1. Tujuan
Membuat website portofolio IT Support yang:
- Profesional & modern
- Mudah dibaca recruiter
- Menampilkan kemampuan real (bukan teori)
- Menggunakan HTML semantic structure
- Mengikuti prinsip RTCCO

## 2. Core Identity
Role: IT Infrastructure & Support  
Positioning: Problem Solver & System Keeper  

Tagline:
"When systems fail, I step in."

## 3. Design Direction
Theme:
- Light (terang)
- Clean & modern
- Tech feel (subtle)

Karakter:
- Sederhana tapi tidak kosong
- Detail kecil (shadow, hover, accent)

## 4. Struktur Halaman
1. Header + Hero
2. Capabilities
3. Experience + Incident Log
4. Case Study
5. Tools & Projects
6. Contact

## 5. Differentiation
- Incident Log (real experience)
- Case Study (thinking process)
- Real tools

## 6. Storytelling
Problem → Analysis → Action → Result

## 7. RTCCO
Readability, Typography, Color, Consistency, Organization

## 8. Prinsip
- Fokus problem solving
- Gunakan semantic HTML
